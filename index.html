Intro

I've lived in many different places, from Florida to New Zealand, and done many different things, including filmmaking, songwriting, motion graphics, and design. Mostly I enjoy creating things (including code!). I've recently started delving into Full-stack development and I can't get enough of it. I enjoy working with Ruby on Rails and front-end technologies like Angular and Ember.js. Mostly I enjoy learning new stuff about web development and adding skills to my arsenal.


There's a certain sense of accomplishment and satisfaction that comes with crafting something. I've always been fascinated by the connection between science and art, the balance of technique and creative vision. The duality in those seemingly opposite spectrums, like yin and yang, come together in a beautiful way, complimenting each other. In any case, at some point I began to feel an interest in the art and science of coding. How far down the rabbit hole will I go? It's a long way down...

Portfolio

Bluebird Feeder

An RSS reader with emphasis on content discovery and feed organization. Created with Ruby on Rails and EmberJS and deployed using Capistrano to a Digital Ocean vps.

Bookclipper

Bookclipper is a social bookmarking site built with Rails as an API and using Ember-cli for the front-end. It also features oauth integration with Twitter.

Grasshopper Boards

An open message-board API. The API in written in Rails and consumed with an AngularJS client. It includes elastic search integration and token-based authentication from scratch.

TribalDev

A community blog focused on learning. Built with Ruby on Rails, the site has many typical blogging features, pagination with infinite scroll, and search integration.

Simple Task

A to-do app which self-destructs tasks older than 7 days. Simple Task was built with Rails 4 and uses AJAX extensively to provide a more seemless user experience.

Codeca

Built with Ruby on Rails as a social coding platform with reddit-like implementations. It features posts with voting, commenting, image uploading, built-in markdown formatting, and infinite scroll.
